Getting start

Follow the instructions below for enjoy of LUANPA-CSS.
1. Donwload the luanpa-css folder.
2. Unzip the folder and save the file "luanpa-css.min.css" to a directory within your project.
3. Add the link tag in the head of your html file and enjoy of LUANPA-CSS
Copy to clipboard <link rel="stylesheet" href="directoryOfYourStyles/luanpa-css.min.css"/>
